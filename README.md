## 빌드 및 실행 방법
  * Java, maven은 설치되어 있어야 합니다.
  * 프록젝트 maven으로 빌드(프로젝트 디렉토리에)
  ```bash
  $ mvn install
  ```
  * 빌드 후 java -jar 으로 실행 방법 
  ```bash
  $ java -jar target/house-0.0.1-SNAPSHOT.jar
  ```
  * maven으로 spring boot 실행 방법
  ```bash
  $ mvn spring-boot:run
  ```
